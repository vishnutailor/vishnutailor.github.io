# Setup instructions

## Part 1. Create your GitHub account

1. Go to https://github.com/signup
2. Use a professional username such as vishnutailor or vishnu-tailor.
3. Verify your email address.
4. Add your real name and a simple profile photo.
5. In your profile bio, paste this sentence.

Incoming UC San Diego economics student interested in finance, entrepreneurship, and technology.

## Part 2. Publish the personal website

1. In GitHub, click the plus sign in the top right corner.
2. Click New repository.
3. Name the repository YOURUSERNAME.github.io. Replace YOURUSERNAME with your exact GitHub username.
4. Set the repository to Public.
5. Click Create repository.
6. Click Add file.
7. Click Upload files.
8. Upload index.html from this folder.
9. Click Commit changes.
10. Open https://YOURUSERNAME.github.io in a browser. Replace YOURUSERNAME with your exact GitHub username.

Before uploading, open index.html in a text editor and replace these placeholders.

YOUR_LINKEDIN_URL
YOUR_GITHUB_URL
YOUR_EMAIL_ADDRESS

## Part 3. Create your GitHub profile README

1. In GitHub, click the plus sign in the top right corner.
2. Click New repository.
3. Name the repository with your exact GitHub username only.
4. Set it to Public.
5. Check Add a README file.
6. Click Create repository.
7. Open the README file.
8. Click the pencil icon.
9. Delete the default text.
10. Paste the contents of profile-README.md from this folder.
11. Replace YOUR_WEBSITE_URL and YOUR_LINKEDIN_URL.
12. Click Commit changes.

## Part 4. Add a custom domain later

The default site address is enough for tonight.

You can buy a domain such as vishnutailor.com later if it is available. GitHub Pages supports custom domains. Follow GitHub's official custom domain instructions after buying the domain.
